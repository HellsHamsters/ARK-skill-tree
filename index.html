ARK engram calculator

This is a tool for ARK: Survival Evolved. It will help you figure out what to spend your engram points on.

As there are 4194 engram points worth of items, but you are only able to earn 2764 engram points (at level 94), therefore it is impossible to learn everything.

  • If an item requires another item, that item will automatically be selected for you (if you have enough points).
  • When you save, the URL will update so you can send your build to friends.

You can create a Mindwipe Tonic in game if you need to reset your engrams.

The code for this can be found on GitHub, if you want to contribute either raise an issue or make a pull request with your changes.
